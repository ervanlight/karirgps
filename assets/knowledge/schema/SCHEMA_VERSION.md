# SCHEMA_VERSION

## Schema aktif

Knowledge Object V2

## Schema lama

Knowledge Object V1

## Tanggal dibuat

2026-06-28

## Tujuan schema

Menjadi standar struktur knowledge object KarirGPS yang konsisten, machine-readable, human-readable, dan siap digunakan untuk AI Retrieval, RAG, reasoning, recommendation engine, dan knowledge graph.

## Catatan migrasi

- Knowledge Object V1 tetap dianggap sebagai sumber awal yang dapat dibaca manusia.
- Knowledge Object V2 menjadi target struktur baru untuk pengembangan jangka panjang.
- Migrasi dari V1 ke V2 perlu mempertahankan isi substantif, tetapi menambahkan metadata, structured fields, relationship, graph, scoring, dan reasoning fields.
- File jurusan yang sudah ada perlu dipetakan ke schema V2 secara bertahap tanpa menghilangkan konteks penting dari V1.
